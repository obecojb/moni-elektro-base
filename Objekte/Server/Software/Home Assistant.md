Der Home Assistant zeigt alle in in Haus installierten Komponenten und verwaltet sie.

![[homeassistant.webp]]

Die Wlan-, Ethernet- und Internet-Zugänge erfolgen über eine [[Fritz Box]]. Zusätzlich überwacht der Home Assistant den gesamten [[Stromverbrauch]]. Weiterhin ist eine Anbindung zusätzlicher Sensoren über einen [[Zigbee Sonoff Dongle M]] realisiert.
