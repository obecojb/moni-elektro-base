# Cloudflare DNS

## Beschreibung

Cloudflare DNS ist ein schneller und sicherer DNS-Dienst, der für das Haus-Netzwerk verwendet wird. Er bietet Schutz vor Malware, Phishing und unerwünschten Inhalten sowie DNS-over-HTTPS (DoH) für Sicherheit.

## Hersteller-Informationen

- **Unternehmen**: Cloudflare
- **Website**: https://1.1.1.1/
- **Fokus**: Schnelle, sichere und private DNS-Auflösung

## DNS-Server

- **Primary**: 1.1.1.1
- **Secondary**: 1.0.0.1
- **IPv6 Primary**: 2606:4700:4700::1111
- **IPv6 Secondary**: 2606:4700:4700::1001

## Sicherheitsfeatures

- **Malware Protection**: Blockierung bekannter Malware-Domains
- **Adult Content Filtering**: Optional zur Blockierung nicht jugendfreier Inhalte
- **Privacy**: Keine Logging von DNS-Anfragen
- **DNSSEC**: Validierung von DNS-Antworten
